Quick Start

# Get all users
curl http://localhost:5000/api/users

# Create a user
curl -X POST http://localhost:5000/api/users \
  -H "Content-Type: application/json" \
  -d '{"name":"John","email":"john@example.com","role":"Developer"}'

# Test status codes
curl http://localhost:5000/api/status/404

# Upload a file
curl -X POST http://localhost:5000/api/upload/single \
  -F "file=@image.jpg"

Key Endpoints

GET /api/users

Retrieve all users with filtering support

POST /api/users

Create a new user

GET /api/headers

View all request headers

GET /api/status/:code

Test specific HTTP status code

POST /api/upload/single

Upload a single file

GET /api/stream/sse
